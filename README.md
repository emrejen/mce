# MCE

## Install

1.  `git clone https://github.com/emrejen/mce.git`
2.  `cd mce`
3.  `npm i`
4.  `npm run build`
5.  `cd frontend`
6.  `npm i`
7.  `npm run build`
8.  `cd ../`
9.  `npm run start`

##View
Web app
http://localhost:8080/
